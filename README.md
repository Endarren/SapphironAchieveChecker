**Note: Does not currently take racial resistance passives into count.**

This is a simple addon I wrote to check the frost resistance of a raid group, specifically for Sapphiron in Naxx, the The Hundred Club achievement. The reason for this is that in the next expansion, resistances are being removed, so there is a time limit to get that achieve and this addon will make sure a pug group does not mess it up.

To use it, just type "/sac check", and it will check the resistances of all of the raid members and announce them in raid chat.

A second command, "/sac tar" was added. This will do the check on your current target.
